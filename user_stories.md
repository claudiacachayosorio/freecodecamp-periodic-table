# Periodic Table
## User Stories

- **input**: none
    > Please provide an element as an argument.

- **input**: element that doesn't exist as an atomic_number, symbol, or name in the database
    > I could not find that element in the database.

- **input**: 1 or H or Hydrogen
    > The element with atomic number 1 is Hydrogen (H). It's a nonmetal, with a mass of 1.008 amu. Hydrogen has a melting point of -259.1 celsius and a boiling point of -252.9 celsius.

- **input**: other element in database
  --> same output but with information associated with the given element
